### 🤖 AI-Driven Smart AQI & Hygiene Advisory Platform
**Capstone Project**  
*Feb 2026 – Present | Associated with NIIT University*

Air pollution dashboards show how bad the air is, but they rarely tell us what to actually do about it. To address this gap, I developed an AI-Driven Smart AQI & Hygiene Advisory Platform that transforms raw pollution data into actionable intelligence for both individuals and institutions. The system provides real-time AQI monitoring and 1–3 day forecasts using LSTM-based time series models, classifies health risk levels (Low/Medium/High) with Random Forest and XGBoost, and delivers personalized recommendations such as indoor workouts, mask selection, ventilation guidance, and outdoor activity advisories. It also features an institution dashboard to support decisions on sports, assemblies, and safety protocols, along with an AI-powered chatbot and Explainable AI (SHAP) insights that clarify model predictions (e.g., PM2.5 contribution to AQI spikes). Built using Python (FastAPI/Flask), React, REST APIs, JWT authentication, Docker, CI/CD pipelines, and secure API architecture, this project integrates AI, public health, DevOps, and cybersecurity into a fully deployable, real-world decision-support system.

**Tech:** Python · FastAPI/Flask · React · REST APIs · JWT · Docker · CI/CD
